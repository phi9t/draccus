* Retrospective
- Bare =pip=/=pip3=: no false-positive in Draccus itself (shim exits 2 with a deliberate message); any upstream script inside the sandbox that invokes plain =pip install= must switch to =draccus-uv pip=. Spack/py-pip’s real binaries remain reachable only off-shim PATH or via =python -m pip=.
- =python -m pip=: still the expert hatch (not blocked intentionally). Enforcement remains Gate 10b on project lockfiles and runtime overlays, not shim import interception.
- Pin bump ergonomics: release =.sha256= sidecar plus tarball URL pattern worked on first bootstrap after implementation; tarball layout favors top-level =uv= (fallback =find -print -quit=). Idempotent skip uses =uv --version | grep VERSION= rather than hashing the unpacked binary vs tarball digest.
